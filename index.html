ParanoidPackrat by jeffWelling

Backup script for *nix systems, using Ruby, for configurable redundancy across >=1 physical device

Paranoid Packrat is a Ruby program to help you backup multiple directories and files, to either one or more hard drives.

Features

1. Configurable Redundancy for multiple hard drives

2. Backups stored in human browse-able directory structure

3. Encrypt your backups with your GnuPG

4. Backup to a drive, disconnect the drive

5. Avoid unnecessary duplication

For more information, see doc/index.html in your ParanoidPackrat directory, and the wiki

Dependencies

Ruby

Install

git clone git://github.com/jeffWelling/ParanoidPackrat.git

cd ParanoidPackrat

cp ParanoidPackrat.default.config.rb ParanoidPackrat.config.rb

Edit ParanoidPackrat.config.rb to configure your desired backups

./ParanoidPackrat.rb
